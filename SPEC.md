# 冷蔵庫食材管理・賞味期限トラッカー｜食品ロスをゼロにする無料アプリ - 技術仕様書

## 概要

**サービス名**: Fridge Tracker
**バージョン**: 1.5.1
**更新日**: 2026-06-20
**URL**: https://appadaycreator.com/fridge-tracker/

冷蔵庫の食材を無料で管理。賞味期限が近い食材を自動警告。食材から使えるレシピ候補を提案。買い物リスト生成機能付き。登録不要でスマホに保存。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 「食材を追加」ボタンからカテゴリ・食材名・数量・賞味期限・保存場所を登録
2. 食材カードをタップして編集、×ボタンで削除
3. 検索ボックスで食材名を絞り込み（リアルタイム）
4. 統計バーのフィルターで「期限切れ」「まもなく期限」「余裕あり」を切り替え
5. レシピタブで冷蔵庫の食材から作れるレシピを確認。「🛒 不足分を追加」で買い物リストに自動追加
6. 期限切れフィルター中に「全件を買い物リストへ」で一括追加

## よくある質問（FAQ）

**Q: 冷蔵庫食材管理・賞味期限トラッカー｜食品ロスをゼロにする無料アプリは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
